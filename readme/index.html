Webservice object generator

ความสามารถ

แปลง Webservice string เดิมๆ ที่คั่นด้วย | มาเป็น JSON Object

วิธีการใช้งาน

  1. สร้างไฟล์ json ไว้ภายใต้ path blueprints โดยมีรูปแบบดังนี้ 
                
                {
                    "serviceName": "tabst",
                    "serviceNumber": "1004",
                    "startAt": "2",
                    "parameters": [
                        "flagOrder"
                    ],
                    "properties": [
                        "natCode",
                        "natDesc"
                    ]
                }

    **ชื่อไฟล์จะใช้ convention เดียวกันกับการตั้งชื่อ class เช่น service ที่ใช้ดึงรายการสัญชาติ ก็จะตั้งชื่อเป็น Nationality.json

    serviceName : หมายถึงชื่อ service โดยละคำว่า sdpa_ ทิ้งไป

    serviceNumber : หมายถึง หมายเลข service

    startAt : หมายถึง จุดเริ่มของข้อมูล (นับ pointer แบบ array)

    parameters : list ของ parameters ตอนส่งไป get ข้อมูล (จำเป็นต้องใส่เรียงตาม sr)

    properties : list ของ properties ตอนได้รับข้อมูลจาก service (จำเป็นต้องใส่เรียงตาม sr)

  2. จากนั้นไปที่ url:webservice/compile เพื่อ compile webservice
  3. ไปที่ url:webservice/services/getNationality.php?flagOrder=1 เพื่อเรียกใช้งาน webservice

Enjoy!!
